Membuat "Hello World"

Untuk memuat tulisan, kita bisa menggunakan tag < p >, ataupun kita bisa menggunakan tag heading. pada materi pertama ini kita akan membuat "hello world" menggunakan tag < p >.

Maka untuk memebuat tulisan hello world cukup tulis dengan kode berikut: < p > Hello World < /p > lalu tambahkan pagargaf baru untuk menuliskan Selamat datang di percobaan pertama dengan kode < p > Selamat datang di percobaan pertama maka hasilnya seperti dibawah ini:

Hello World

Selamat datang di percobaan pertama
